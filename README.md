# vimail - A command line tool for personalized bulk emailing
A handy little script i am developing to solve a very specific problem at work, here's hoping it might be of use to someone else too.

# Features
Allows you to send emails based on a template to many addresses quickly, giving you the time to make unique inclusions and add unique attachments to each address.
Currently supports Gmail and Outlook.

# Instalation
- git clone https://github.com/Mechaspirit1/vimail
- cd vimail
- mv vimail.py vimail & mv vimail /usr/local/bin

It is important to note that you will need an App password from your email provider to make this work.
